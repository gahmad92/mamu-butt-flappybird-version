# MAMU BUTT ADVENTURE: PLANNER

## 🌟 Vision
A whimsical, Ghibli-inspired "Flappy Bird" clone featuring the legendary character "Mamu Butt" (🧔). The game prioritizes cozy aesthetics, relaxing atmosphere, and accessible gameplay.

## 🛠 Technical Architecture
- **Framework**: React 19 (UI & State Management)
- **Engine**: Phaser 3.90 (Game Logic & Rendering)
- **Physics**: Arcade Physics (Simple gravity and collisions)
- **Communication**: Phaser EventBus (Syncing React UI with Game State)

## 🎨 Design Principles (Ghibli Style)
- **Palette**: Soft pastels, warm whites, sky blues, and mossy greens.
- **Atmosphere**: Peaceful, slightly nostalgic, with environmental details like floating petals and rolling clouds.
- **Gameplay**: "Forgiving" difficulty. Large gaps in hurdles and small player hitboxes.
